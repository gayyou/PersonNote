6. 实现一个flat，数组扁平化，也就是ES6的flat方法，怎么做？

   递归

   ```js
   //line=readline()
   //print(line)
   // 递归，每次进行递归的时候重数-1，然后为0的时候就不扁平化
   const isUndef = tar => typeof tar === 'undefined' || tar === null;
   // flat([1,[2, [3]]], 1) => [1, 2, [3]]
   // flat([1,[2, [3]]])  => [1, 2, 3];
   
   function flat(arr, depth = Infinity) {
       if (depth === 0) {
           return arr;
       }
       let result = [];
       
       for (let item of arr) {
           // 进行遍历，然后遇到数组的时候，进行递归调用
           if (Array.isArray(item)) {
               result = [...result, ...flat(arr, depth - 1)];
           } else {
               result.push(item);
           }
       }
       
       return result;
   }
   ```

   现在延伸出一个问题，函数参数的默认值问题，传入一个undefined的时候，会使用默认值，传入一个null呢？

   答：首先我是没使用过，但是如果我现在想的话，答案是不会的。原因如下：

   首先typeof null  === 'Object'，null其实是一个对象来着

   其次有的函数调用者，有时候传参会传null，所以说这个是一个有意而为之的行为，所以不应当使用默认值
